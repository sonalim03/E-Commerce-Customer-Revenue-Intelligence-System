## 📌 Project Overview
This project analyzes a real-world **Brazilian E-Commerce dataset (Olist)** to uncover customer behavior, revenue trends, and business performance metrics. The goal is to help businesses make smarter, data-driven decisions.

---

## 🔑 Key Insights & Results
| Insight | Finding |
|---|---|
| 💰 Top Revenue Segment | 20% of customers generate 80% of revenue |
| 📦 Best Selling Category | Health & Beauty / Electronics |
| 📈 Peak Sales Period | November (Black Friday season) |
| ⚠️ Churn Risk | 40%+ customers never made a second purchase |
| 🌍 Top Region | São Paulo contributes highest revenue |
| ⏱️ Avg Delivery Time | Impacts customer satisfaction score significantly |
| 🔄 Repeat Purchase Rate | Analyzed using RFM segmentation |

## 🚀 How to Run This Project

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/sonalim03/E-Commerce-Customer-Revenue-Intelligence-System.git
```

### 2️⃣ Download Large Files (Git LFS)
```bash
git lfs install
git lfs pull
```

### 3️⃣ Run Python Notebooks
```bash
pip install pandas numpy matplotlib seaborn scikit-learn
jupyter notebook
```

### 4️⃣ Open Power BI Dashboard
- Open `.pbix` file in **Power BI Desktop**

### 5️⃣ Run SQL Queries
- Open `olist_customer_analytics.sql` in MySQL / PostgreSQL
